*******************************************************
*  Description of the program
*******************************************************

This program creates a linked-list from scratch using the provided LinkedList
header file. A simple demonstration in main shows the correct functionality of
insertFront, insertBack, deleteFront, deleteBack, insertBeforePosition,
deletePosition, clear, and properly implements the clear function within
the LinkedList destructor.

*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program. This is the driver program that demonstrates the correct
   functionality of the LinkedList member functions.

Name:  LinkedList.h
   Contains the definition for the class LinkedList.

Name: LinkedList.cpp
   Defines the default constructor, and the functions clear, insertFront,
   insertBack, deleteFront, deleteBack, insertBeforePosition, deletePosition,
   clear, and the destructor.
