*******************************************************
*  Description of the program
*******************************************************

This program creates a vector from scratch using the MyVector class. A simple
demonstration in main shows the correct functionality of the functions empty(),
push_back, pop_back, and search. Included is analysis.txt which is a partial
asymptotic analysis of functions using Big O.


*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program.  This is the driver program that demonstrates the correct
   functionality of the MyVector member functions.

Name:  MyVector.h
   Contains the definition for the class MyVector.

Name: MyVector.cpp
   Defines the default constructor, and the functions empty(), push_back,
   pop_back, search, and [] operator overload.
